# BeeMarkdown
简单轻巧的Markdown文档在线预览工具.

## Useage

###### 启动md服务
``` bash
md start [port]端口号,默认2222
```

###### 初始化md
``` bash
md init
```


###### 其它命令
```shell
md --help     (显示帮助信息)
md --version  (显示md版本号)
```
